Message-Driven Application - User Guide

Overview

This application simulates a message-driven system using a producer-consumer pattern in Core Java.

Components

  • Message Class - Represents individual messages with an ID and content.
  • Producer Class - Generates messages and places them in a queue.
  • Consumer Class - Consumes messages and logs processing results.
  • Main Application - Manages execution flow using ExecutorService.

Application Workflow

  1. The producer generates messages and adds them to the queue.
  2. The consumer retrieves messages and processes them.
  3. Processing fails for some messages to simulate errors.
  4. The consumer logs success and failure counts.

Running the Application

Prerequisites

  • Java 8 or later
  • IntelliJ IDEA (or any Java IDE)
  • JUnit & Mockito (for testing)

Steps

  1. Extract the MessageQueueApp.zip file.
  2. Open the project in IntelliJ IDEA.
  3. Run the main method in MessageQueueApp.java.
  4. Execute unit tests to validate functionality.
